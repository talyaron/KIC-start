# Live Color Sync App

A real-time web application where users can change the background color, and it updates instantly for everyone connected.

## Features
- **Real-time Sync**: Uses Firebase Firestore to listen for color changes.
- **Premium UI**: Glassmorphism design, smooth transitions, and responsive layout.
- **No Build Step**: Built with vanilla HTML/JS/CSS for easy deployment and testing.

## Setup
1.  Open `index.html` in your browser.
2.  Enjoy!
