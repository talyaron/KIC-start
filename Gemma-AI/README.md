# 🤖 Dan - AI Chatbot (Python + Gemma 3)

בוט צ'אט מבוסס בינה מלאכותית בפייטון עם Ollama ו-Gemma 3

## 🚀 התקנה מהירה

### 1️⃣ התקן Ollama

**Windows:**
- הורד מ-https://ollama.com/download
- הרץ את קובץ ההתקנה
- עקוב אחר ההוראות

**Mac/Linux:**
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### 2️⃣ הורד את Gemma 3

```bash
ollama pull gemma3
```

או בחר גרסה ספציפית:
```bash
ollama pull gemma3:1b   # מהיר (1B פרמטרים)
ollama pull gemma3:4b   # מאוזן (4B פרמטרים) - מומלץ
ollama pull gemma3:12b  # חזק (12B פרמטרים)
ollama pull gemma3:27b  # מתקדם (27B פרמטרים)
```

### 3️⃣ התקן תלויות Python

```bash
pip install -r requirements.txt
```

או ידנית:
```bash
pip install requests
```

### 4️⃣ הרץ את הבוט!

```bash
python chatbot.py
```

## 🔧 פתרון בעיות

### הבוט לא מתחיל
```bash
# בדוק אם Ollama מותקן
ollama --version

# בדוק אם Gemma הורד
ollama list

# הפעל את שרת Ollama
ollama serve
```

### שגיאת חיבור
- ודא ש-Ollama פועל ברקע
- בדוק שהשרת זמין ב-`http://localhost:11434`
- נסה להפעיל מחדש את Ollama

### המודל איטי
- נסה מודל קטן יותר: `gemma3:1b`
- סגור אפליקציות אחרות
- בדוק את משאבי המחשב

### שגיאת Python
```bash
# ודא שהגרסה נכונה
python --version  # צריך להיות 3.7+

# התקן מחדש את התלויות
pip install --upgrade requests
```
